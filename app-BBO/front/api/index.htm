The purpose of this sample application is to demonstrate an application using the Livebookings External Web Service API using Microsoft .NET.

The first thing to do is to configure your partnercode, and languages you wish to use in the SoapClient.php file. Please set $mPartnerCode to the partner code you have been assigned by Livebookings. You may also change the language you use in this file.

In this sample. You will be able to search for restaurants content in the "Content Sample". You can also make a booking to our test restaurant "Eat Food" in the "Book Sample". After you have made a booking, you can cancel the reservation in the "Cancel Sample". Finally, you can access the promotions content in the "Promotion Sample".
